# 🧠 Medical Image Classification using CNN

## 📌 Overview

This project applies Deep Learning (Convolutional Neural Networks - CNN) to classify chest X-ray images into two categories:

* NORMAL
* PNEUMONIA

The model is trained on a real medical dataset and demonstrates the application of AI in healthcare diagnostics.

---

## 🚀 Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy

---

## 📊 Model Performance

* Training Accuracy: ~92%
* Validation Accuracy: ~75%

---

## 🧪 Features

* Image preprocessing
* Data augmentation
* CNN-based classification
* Prediction on new images
